# Redux based Todo app

In this project we are using the latest redux version

## Packages :

- tailwind

```
yarn add tailwind
```

- uuidv4

```
yarn add uuidv4
```

- redux

```
yarn add redux
```

- redux/toolkit

```
yarn add yarn add redux-toolkit
```

- redux-logger

```
yarn add redux-logger
```

- redux-devtools

```
yarn add redux-devtools-extension
```

- react-redux

```
yarn add react-redux
```
